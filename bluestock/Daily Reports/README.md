# Daily Reports

This folder contains daily progress reports organized by month.

## Structure
- **September/** - Daily reports for September 2025
- **October/** - Daily reports for October 2025

## Report Format

Each daily report should include:
- Date
- Tasks completed
- Issues encountered
- Time spent
- Next steps

## Naming Convention

Please name reports as: `YYYY-MM-DD_Report.md`

Example: `2025-09-15_Report.md`
